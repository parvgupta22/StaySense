# 📖 Overview

Traditional Airbnb analytics often compare listings against broad
averages, producing misleading insights.

StaySense benchmarks listings against genuinely comparable peers using
hierarchical filtering, similarity search and custom KPIs to provide
meaningful performance insights.

# 🏆 Custom KPIs

## ⭐ StayScore

StayScore is an overall performance metric that evaluates an Airbnb listing by combining multiple factors such as guest ratings, pricing competitiveness, host reliability, and listing quality. It provides a single score that helps benchmark listings against comparable properties.

---

## 🤝 HostTrust

HostTrust measures the reliability of a host based on factors such as response rate, response time, acceptance rate, Superhost status, and review history. A higher HostTrust score indicates a more dependable host experience.

---

## 🏡 ListingIQ

ListingIQ evaluates the overall quality and competitiveness of a listing by considering attributes such as amenities, photos, property details, occupancy-related indicators, and pricing relative to similar listings.

---

## 😊 ExperienceIQ

ExperienceIQ reflects the expected guest experience by combining review ratings, cleanliness, communication, location, value, and other guest satisfaction indicators into a single benchmark score.
